### A simple TDD lab-example using Laravel HTTP Tests API

**Tests**
```
Article Domain:
  ✓ should return articles on correct data format
  ✓ should create article successfully
  ✓ should return json with validation error message on article store
  ✓ should return article by id
  ✓ should return model not found when passing article invalid id
  ✓ should update created article infos
  ✓ should return json with validation error message on article update
  ✓ should delete article
  ✓ should return model not found when passing article invalid id on destroy
```
